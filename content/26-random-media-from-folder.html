Random Media From Folder

What it is: A Streamer.bot action that picks a random file from a folder. You pass the folder path and the type (video, image, music, or text); the action selects one matching file and sets the full path in the argument randomFile so other actions (e.g. OBS media source, play sound) can use it.

Action arguments

  • folderPath — Full path to the folder to scan (required).
  • fileType — One of: video, image, music, text (required). The action only considers files with matching extensions (e.g. .mp4, .png, .mp3, .txt).

Output: randomFile — full path to the chosen file.

How to set up

Import the action (Copy / Download / Open below) and pass folderPath (full path to a folder) and fileType (video, image, music, or text). Chain it with an OBS media source or play-sound action that uses the randomFile output. Add assets/images/page-images/random-media-setup.png for a screenshot.

Examples

E.g. random outro video, random sound effect, or random image for a scene. Add assets/images/page-images/random-media-example.png to show your workflow.

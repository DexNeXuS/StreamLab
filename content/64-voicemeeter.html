VoiceMeeter

Virtual audio mixer and routing so game, music, mic, and alerts all land in the right place. Virtual cables and strips so the stream hears exactly what you want — and you can monitor without feedback.

Official: VoiceMeeter (Potato) — other variants (Standard, Banana) on the same site.

How to set up

Install VoiceMeeter (Standard, Banana, or Potato depending on your I/O needs). Set it as the default playback/recording device where needed, and route OBS to capture the VoiceMeeter output. Optional screenshot: assets/images/page-images/voicemeeter-setup.png.

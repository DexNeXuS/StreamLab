Streamer.bot

CMD CTRL

What it is: A Streamer.bot action that turns things on or off by name or ID: rewards, commands, command groups, timers, or actions. Pass the type, the id/name (or several separated by ||), and enabled true/false. Ideal for Touch Portal (or other triggers) to enable/disable whole groups or single commands without opening Streamer.bot.

Action arguments

  • id — Command/reward/timer/action ID (GUID) or name. Multiple items: separate with || (e.g. MyCommand||OtherCommand).
  • type — One of: reward / rewards, command / cmd, commandgroup / group, timer / timers, action / actions.
  • enabled — true to enable, false to disable.

Example (Touch Portal): call this action with type=command, id=!heal, enabled=false to turn off the !heal command.

How to set up

Import the action (Copy / Download / Open below), then from Touch Portal (or another trigger) call it with type, id, and enabled. E.g. disable a command by name, or toggle a command group. Add assets/images/page-images/cmd-ctrl-setup.png for a screenshot.

Examples

Turn off !heal during a segment, or enable/disable a whole group of commands. Add assets/images/page-images/cmd-ctrl-example.png to show your Touch Portal layout or Streamer.bot bind.

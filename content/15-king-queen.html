Overlay + Streamer.bot

King-Queen overlay

What it is: A two-panel “King” and “Queen” overlay for OBS that listens for WebSocket messages from Streamer.bot. Use it for polls, duels, or any two-option display. You add the overlay as a Browser Source in OBS, then use the Streamer.bot action below to send pings (or other commands) to control it.

Overlay URL (OBS Browser Source)

In OBS, add a Browser Source and set the URL to the address below. This is the same URL shown in the Overlays list on the OBS page — you can use either place to copy it.

Streamer.bot action import

The action sends a WebSocket message to the overlay (e.g. to trigger or test the display). Use the box below to Copy, Download, or Open the import file, then in Streamer.bot go to Import → Import from clipboard.

How to set up

Add a Browser Source in OBS with the overlay URL above. In Streamer.bot, import the action and trigger it (e.g. from a command or button) to send the WebSocket message that controls the overlay. Adjust these steps to match your workflow.

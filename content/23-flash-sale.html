Flash Sale

What it is: A Streamer.bot action that turns any channel point reward into a flash sale: when triggered, it temporarily lowers the cost of that reward (or multiple rewards) by a percentage, then restores the original prices after a set time. Great for "flash sale" redemptions on Twitch.

Action arguments (variables)

When you trigger the action (e.g. from a channel point reward or Touch Portal), pass these so the action knows how to run the sale:

  • percentOff — Discount percentage (e.g. 50 for 50% off). Default 99 if not set.
  • saleTimer — How long the sale lasts, in seconds (e.g. 60). Default 30.
  • cpPrice — If set, use this as the fixed channel point price during the sale instead of a percentage off.
  • fairPrice — Minimum/fair price in channel points when using percentage off. Default 500.

The action stores original prices, applies the sale, starts a timer, and when the timer fires it restores all rewards to their original costs.

How to set up

Create a channel point reward in Twitch, then in Streamer.bot link that reward to this action. Set the action arguments (percentOff, saleTimer, etc.) when you trigger it — e.g. from the reward or from Touch Portal. Add a setup screenshot as assets/images/page-images/flash-sale-setup.png if you want.

Examples

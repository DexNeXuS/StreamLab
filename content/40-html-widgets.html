Browser-source widgets

Ready-to-use widgets for OBS Browser Source. Use Copy URL and paste into OBS (add ?param=value for options where supported), or Download and edit the HTML locally. Each card has a How to link when instructions are available.

How to use

  • Copy URL — Paste the URL into OBS: Add Source → Browser → paste. Many widgets support URL arguments (e.g. ?effect=typewriter&lines=Line1,Line2). Check the widget’s How to for parameters.
  • Download — Save the HTML file and point OBS to the local file, or edit the file to change text, timings, or styles.
  • How to — Opens the widget’s how-to guide in a new tab (params, examples, and setup).

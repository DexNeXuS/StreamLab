OBS + RockSniffer

Rocksmith widget

What it is: A browser-source overlay for OBS that shows your current Rocksmith 2014 session: now playing (song, artist, tuning, progress), performance (score, accuracy, notes), streak, accuracy, and optional session/custom cards. It polls RockSniffer (local HTTP) for live stats and can receive updates from Streamer.bot via a global variable or WebSocket.

RockSniffer: nickspooner/rocksniffer — run it locally so the overlay can read session data.

Overlay URL (OBS Browser Source)

In OBS, add a Browser Source and set the URL below. You can add query params for position, style, and RockSniffer/Streamer.bot (e.g. ?pos=br&style=narrow&rsPort=9938).

How to use & control

Full instructions: RockSniffer, Streamer.bot (global variable + WebSocket), URL parameters, toasts, and optional Widget → Streamer.bot.

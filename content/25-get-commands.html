Streamer.bot

Get Commands

What it is: Two Streamer.bot actions that work together: (1) Make JSON exports your commands into a JSON file with optional descriptions and permissions per command. (2) Run reads that JSON and posts a formatted command list to chat. You can set a custom icon/prefix and title per group, show or hide descriptions, and filter by permissions (e.g. show only mod commands). Great for “!commands” style messages and for Touch Portal buttons that type or post command lists.

How to set up

Run the “Make JSON” action once to export your commands to commands.json. Set the “Run” action’s jsonPath and cmd (group name). Trigger “Run” from a chat command, Touch Portal, or channel point reward. Add a screenshot as assets/images/page-images/get-commands-setup.png if you want.

JSON format (from Make JSON)

The export produces a file like commands.json with:

  • groups — One key per Streamer.bot command group.
  • Each group has icon (emote/prefix), title (custom name; if empty, group name + “Commands” is used), and commands.
  • commands — For each command (by primary trigger): description (optional), permissions (e.g. "" for everyone, "Mod" / "Sub" / "VIP" for restricted).

Run the “Make JSON” action whenever you add/remove commands; edit the JSON to add descriptions or permissions. The “Run” action uses this file to build the chat message.

Run action arguments

  • cmd — Command group name (required).
  • jsonPath — Full path to the folder or file containing commands.json (required).
  • desc — true to include descriptions in the chat output.
  • sd — true to include disabled commands.
  • alias — true to show aliases.
  • permissions — If set (e.g. Mod), only commands with that permissions value in the JSON are shown (for mod-only lists).

Action import

One import contains the Get Commands plugin (Make JSON + Run). Use the box below to Copy, Download, or Open the import file — or use the list on the Streamer.bot page (Action imports tab). Then in Streamer.bot: Import → Import from clipboard.

Reference scripts

Download the C# scripts (for reference or customisation):

  • make-JSON.cs — export commands to JSON
  • Run.cs — read JSON and post to chat

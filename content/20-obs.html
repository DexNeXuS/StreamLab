How to set up

Basic setup: video capture for your camera, game or window capture for gameplay, and audio capture for mic and desktop sound. Add browser sources for overlays using the URLs from the Overlays tab below. The real magic is in plugins — check the OBS plugin forum and experiment. OBS Scripts

A list of scripts I use or recommend, with short notes.

  • Horaiken3DPanel — scene/control helper panel.
  • Countdown.lua — simple countdown timer.
  • Media_Hide — hide/restore media sources.
  • obs-zoom-to-mouse.lua — zoom around cursor.
  • Source Search Helper — quickly find sources.
  • source-toggler — toggle sources with hotkeys/logic.
  • whiteboard.lua — draw/annotate on stream.

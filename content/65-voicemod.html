VoiceMod

Real-time voice changer and effects. When you want to sound different on stream — or just a bit more fun — VoiceMod sits in the chain between mic and OBS (usually after VoiceMeeter).

Official: VoiceMod — download, installation, and guides.

How to set up

Install VoiceMod and set it as the input for OBS (or as the input to a VoiceMeeter strip). Pick your default effect or hotkeys. Optional screenshot: assets/images/page-images/voicemod-setup.png.

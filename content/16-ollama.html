Ollama Run (INI config)

What Ollama is: Ollama runs large language models (LLMs) locally on your machine. You need Ollama installed and running (with a model pulled, e.g. ollama run llama3.2) for this action to work.

This page provides a Streamer.bot action that calls Ollama for chat-style replies. All options (model, system prompt, reply chance, etc.) are read from an INI file. After importing, set the action’s iniPath argument to the full path of your ollama.ini.

Official: Ollama — download and run LLMs locally.

Action import

Use the box below to Copy, Download, or Open the import file. In Streamer.bot go to Import → Import from clipboard. After importing, set the iniPath argument to the full path of your ollama.ini file.

How to set up

Install and run Ollama, pull a model (e.g. ollama run llama3.2). Import the action below, then set the iniPath argument to the full path of your ollama.ini. You can add a setup screenshot as assets/images/page-images/ollama-setup.png.

Examples

Use the action from a chat command, timer, or channel point reward. Edit ollama.ini to change model, system prompt, and reply chance. Add an example screenshot as assets/images/page-images/ollama-example.png if you want.
